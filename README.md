# Portfolio App

A personal portfolio app built with Flutter to showcase skills, projects, and contact information.

## Features

*   Clean and responsive UI
*   Sections for:
    *   About Me
    *   Skills
    *   Projects
    *   Contact

## Technologies

*   Flutter
*   Provider (State Management)

## Getting Started

1.  Clone the repository
2.  Run `flutter pub get` to install dependencies
3.  Run `flutter run` to start the app
